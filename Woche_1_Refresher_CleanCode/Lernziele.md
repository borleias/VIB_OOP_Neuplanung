# Lernziele Woche 1: Refresher & Clean Code

Die Studierenden sollen am Ende der ersten Woche:
1.  **C#-Grundlagen reaktivieren:** Wiederholung von Klassen, Methoden, Datentypen und Sichtbarkeiten.
2.  **Bedeutung von Clean Code verstehen:** Erkennen, warum Code nicht nur für Computer, sondern vor allem für Menschen geschrieben wird.
3.  **Namensgebung beherrschen:** Intuitiv verständliche Namen für Variablen, Methoden und Klassen wählen.
4.  **Methoden-Design verstehen:** Wissen, wann eine Methode zu lang ist (Single Responsibility Principle auf Methoden-Ebene).
5.  **Refactoring anwenden:** Ersten "schmutzigen" Code in sauberen Code überführen können.

---

## Ablauf
1.  Kurze Wiederholung (30 Min): Was ist eine Klasse? Was ist ein Interface?
2.  Interaktiver Vortrag (60 Min): Clean Code Prinzipien nach Robert C. Martin.
3.  Praktische Übung (90 Min): Das "Spaghetti-Problem" - Refactoring einer Konsolen-Applikation.
